## ما هو DownTube؟

DownTube هو تطبيق مجاني ومفتوح المصدر يعمل محلياً على جهازك لتحميل فيديوهات يوتيوب مع تضمين الترجمة العربية تلقائياً. التطبيق يعمل بالكامل على جهازك الشخصي مما يضمن الخصوصية التامة وسرعة التحميل، ولا يرسل أي بيانات لأي خادم خارجي.

---

## المميزات

- **ترجمة عربية إجبارية** - يبحث تلقائياً عن الترجمة العربية اليدوية أولاً، ثم التلقائية، ويضمّنها في الفيديو
- **واجهة ويب عربية** - واجهة رسومية احترافية تعمل في المتصفح بتصميم داكن أنيق
- **وضع سطر الأوامر** - للاستخدام السريع من التيرمينال
- **تضمين الترجمة** - يدمج الترجمة داخل ملف MKV مع حفظ نسخة SRT منفصلة
- **أفضل جودة** - يحمل بأعلى جودة متوفرة (فيديو + صوت) ويحول لصيغة MKV
- **تقدم لحظي** - شريط تقدم مفصّل مع السرعة والوقت المتبقي وحالة الترجمة
- **إلغاء التحميل** - إمكانية إيقاف التحميل في أي وقت
- **دعم الكوكيز** - يدعم ملفات الكوكيز لتجاوز حظر يوتيوب في بعض الشبكات
- **مجاني بالكامل** - لا يحتاج اشتراك أو تسجيل أو بيانات دفع

## التثبيت السريع

### Linux / macOS

```bash
# 1. استنسخ المستودع
git clone -b cloud-deployment https://github.com/mohamedhaithamx/DownTube.git
cd DownTube

# 2. شغّل سكريبت التثبيت
chmod +x setup.sh
./setup.sh
```

أو يدوياً:

```bash
# تثبيت المتطلبات
pip install -r requirements.txt
pip install yt-dlp

# تأكد من وجود ffmpeg
ffmpeg -version
```

### Windows

```batch
:: 1. حمّل المستودع وفك الضغط
:: 2. انقر مرتين على setup.bat
```

أو يدوياً:

```batch
pip install -r requirements.txt
pip install yt-dlp
```

**ملاحظة لـ Windows:** تأكد من تثبيت ffmpeg وإضافته لمسار النظام (PATH). يمكنك تحميله من [هنا](https://www.gyan.dev/ffmpeg/builds/).

## طريقة التشغيل

### الواجهة الرسومية (ويب)

```bash
python app.py
```

سيفتح المتصفح تلقائياً على العنوان `http://localhost:8555`

**خيارات إضافية:**

```bash
# تحديد منفذ مختلف
python app.py --port 9000

# عدم فتح المتصفح تلقائياً
python app.py --no-browser
```

### سطر الأوامر (CLI)

```bash
# تحميل فيديو
python app.py --cli "https://www.youtube.com/watch?v=VIDEO_ID"

# تحديد مجلد التحميل
python app.py --cli "https://www.youtube.com/watch?v=VIDEO_ID" -o ~/Videos

# تحميل محاضرة MIT (ساعة كاملة) مع ترجمة عربية
python app.py --cli "https://www.youtube.com/watch?v=LECTURE_ID"
```

## كيفية الاستخدام

### عبر الواجهة الرسومية

1. شغّل التطبيق: `python app.py`
2. الصق رابط يوتيوب في خانة الإدخال
3. اضغط على 🔍 لفحص الفيديو (اختياري - يعرض معلومات الفيديو والترجمة)
4. اضغط **تحميل الفيديو مع الترجمة العربية**
5. تابع التقدم لحظياً - شريط التقدم، السرعة، الوقت المتبقي، وحالة الترجمة
6. عند الاكتمال، ستجد الملفات في مجلد التحميل الافتراضي

## آلية عمل الترجمة العربية

يعمل DownTube بستراتيجية ذكية لضمان الحصول على الترجمة العربية:

```
1. فحص الفيديو → هل توجد ترجمة عربية يدوية؟
   ├─ نعم → تحميل الفيديو مع الترجمة اليدوية ✅
   └─ لا → هل توجد ترجمة عربية تلقائية؟
       ├─ نعم → تحميل الفيديو مع الترجمة التلقائية ✅
       └─ لا → تحميل الفيديو بدون ترجمة ⚠️
```

**بعد التحميل:**
- الترجمة تُدمج داخل ملف MKV (يمكن تشغيلها/إخفاؤها في أي مشغل)
- ملف SRT منفصل يُحفظ بجانب الفيديو للاستخدام الخارجي

**أنواع الترجمة:**
| النوع | الوصف | الأولوية |
|-------|-------|----------|
| يدوية | ترجمة رفعها أشخاص - دقيقة ومطابقة | 🔴 أعلى |
| تلقائية | ترجمة autogenerated من يوتيوب - جيدة لكن قد تحتوي أخطاء | 🟡 ثانوية |
| غير متوفرة | لا توجد ترجمة عربية - يُحمّل الفيديو بدون ترجمة | ⚪ لا ترجمة |

## حل مشكلة "يوتيوب حظر الطلب"

إذا ظهرت رسالة **"Sign in to confirm you're not a bot"** أو **"429 Too Many Requests"**، فهذا يعني أن يوتيوب حظر طلباتك مؤقتاً. الحلول:

### 1. استخدام كوكيز المتصفح (الأفضل)

صادِر كوكيز يوتيوب من متصفحك واحفظها في ملف:

```bash
# باستخدام إضافة المتصفح
# ثبّت إضافة "Get cookies.txt LOCALLY" على Chrome/Firefox
# افتح youtube.com ثم صدّر الكوكيز

# احفظ الملف باسم cookies.txt في مجلد التطبيق
mv ~/Downloads/cookies.txt ./cookies.txt

# شغّل التطبيق - سيستخدم الكوكيز تلقائياً
python app.py
```

أو احفظ الكوكيز في:
- مجلد التطبيق: `cookies.txt`
- المجلد الرئيسي: `~/cookies.txt`
- المسار القياسي: `~/.config/yt-dlp/cookies.txt`

### 2. انتظر قليلاً

عادةً الحظر مؤقت ويُرفع بعد 10-30 دقيقة.

### 3. حدّث yt-dlp

```bash
pip install --upgrade yt-dlp
```

## الصيغ المدعومة

| العنصر | الصيغة | السبب |
|--------|--------|-------|
| الفيديو النهائي | MKV | يدعم تضمين الترجمات والحفاظ على الجودة |

## الأسئلة الشائعة

### هل التطبيق مجاني؟
نعم، التطبيق مجاني تماماً ومفتوح المصدر. لا يحتاج اشتراك أو تسجيل.

### هل يعمل على أي جهاز؟
نعم، يعمل على Linux و macOS و Windows طالما Python و ffmpeg مثبّتان.

### لماذا لا تعمل الترجمة العربية أحياناً؟
بعض الفيديوهات ليس لها ترجمة عربية (يدوية أو تلقائية). في هذه الحالة يُحمّل الفيديو بدون ترجمة ويُخبرك التطبيق بذلك.

### هل يمكنني تحميل قائمة تشغيل؟
حالياً يدعم تحميل فيديو واحد فقط. تحميل قوائم التشغيل قيد التطوير.

### الفيديو بطيء التحميل، ماذا أفعل؟
- تأكد من اتصالك بالإنترنت
- حاول استخدام كوكيز المتصفح (انظر قسم حل المشاكل)
- حدّث yt-dlp: `pip install --upgrade yt-dlp`

### كيف أشغّل الترجمة في الفيديو؟
في معظم مشغلات الفيديو (VLC, MPV, MPC-HC):
- اضغط على زر الترجمة (Subtitle) أو مفتاح `V`
- اختر مسار الترجمة العربية
- إذا كانت الترجمة مضمّنة في MKV، ستظهر تلقائياً
