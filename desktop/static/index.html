ملفات تعريف الارتباط (الكوكيز)

غير نشطة

الكوكيز ترفع حد التحميل بشكل كبير جداً. استخدم إضافة "cookies.txt export" من Chrome Web Store لتصدير الكوكيز من يوتيوب بعد تسجيل الدخول.

أو
طريقة تصدير الكوكيز
  1. ثبّت إضافة "cookies.txt export" من Chrome Web Store
  2. افتح youtube.com وسجّل الدخول بحسابك
  3. اضغط على أيقونة الإضافة في شريط المتصفح
  4. اختر "Copy" لنسخ المحتوى أو "Export" لتحميل الملف
  5. الصق المحتوى في الحقل أعلاه أو ارفع الملف
